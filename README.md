# 🚀 VNX - Vietnamese Address Converter

A modern web application for converting Vietnamese administrative addresses between old and new systems following the 2025 administrative reforms.

## 📋 Overview

From July 1, 2025, Vietnam implemented major administrative reforms, merging provinces, renaming cities, and dissolving districts. **VNX** helps users seamlessly convert addresses between the old system (pre-2025) and new system (post-2025).

## ✨ Features

- 🔄 **Bidirectional Conversion**: Convert addresses from old → new and new → old
- 🎯 **Precise Mapping**: Accurate province, district, and ward level conversions
- 🌙 **Dark Mode**: Beautiful light/dark theme toggle
- 📱 **Responsive Design**: Works perfectly on desktop and mobile
- ⚡ **Fast & Lightweight**: Built with Go backend and vanilla JavaScript
- 🎨 **Modern UI**: Clean, accessible interface with smooth animations

## 🛠️ Technology Stack

- **Backend**: Go with net/http
- **Frontend**: Vanilla JavaScript + HTML5 + CSS3
- **Styling**: CSS Custom Properties (CSS Variables)
- **Data**: JSON-based administrative mappings
- **Deployment**: Static hosting (GitHub Pages)

## 📊 Data Sources

Administrative data is sourced from official Vietnamese government publications and updated regularly to reflect the latest changes.
